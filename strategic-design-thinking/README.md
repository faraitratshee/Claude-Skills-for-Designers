# Strategic Design Thinking

![Strategic Design Thinking cover](assets/cover.png)

A Claude skill that turns vague briefs, business challenges, and creative requests into structured Strategic Design Thinking outputs — so every solution addresses the real problem before execution begins.

## What it does

Instead of jumping straight to a solution, this skill routes any brief through six phases:

1. **Empathize** — understand who's affected and why
2. **Define** — surface the root problem behind the surface-level request
3. **Ideate** — generate safe, bold, and disruptive ideas
4. **Prototype** — make the idea tangible
5. **Test** — validate assumptions before launch
6. **Implement** — plan the rollout, KPIs, and risks

Every response is checked against a Creative Director Review Framework (Business, Audience, Brand, Creative, Execution) before it's considered done.

## When it triggers

Use it whenever you're handed:

- A business challenge or marketing brief
- A campaign, branding, or product/service request
- A UX/UI or customer experience problem
- A digital transformation project
- A vague "give me some ideas" ask

## Install

Download `SKILL.md` from this folder and add it to your Claude skills directory, or point Claude at this repo.
